Disappearing Modes Test

Mode 1: Pull & Delete

Characters delete from the beginning with visual effects flying around.

How it works: Text flows in a single line. Characters automatically delete from the left with visual effects. Keeps you focused on the present moment.

Mode 2: Fade in Place

Words fade away in place. Text flows continuously forward, wrapping after 3 lines.

How it works: Type continuously. Words fade away where they are, leaving empty space. Text flows forward across 3 lines, then wraps back to the top.

Mode 3: Burn Away

Characters dissolve from random positions like burning paper.

How it works: Single line of text. Random characters burn away with ember effects, creating organic deletion patterns.
